# tqq-ui 组件库
https://tqq123.github.io/tqq-ui

### 快速开始

#### 1.安装

```bash
# npm
npm i tqq-ui

# yarn
yarn add tqq-ui
```

#### 2.引用

```javascript
// 全部引入
import TqqUI from "tqq-ui";
import "tqq-ui/dist/css/index.css";

Vue.use(TqqUI);
// 按需引入
import { Card } from "tqq-ui";
import "tqq-ui/dist/css/card.css";

Vue.use(Card);
```
